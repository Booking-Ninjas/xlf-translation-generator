📤 Generate Translation File

ℹ️ How to Use (Full Instructions)

  • Flow 1 — Update Database: Upload XLF from a clean scratch org to sync labels with Google Sheets
  • Edit: Open Google Sheets and add/edit translations for each language
  • Flow 2 — Generate File: Select language, attach source mask, download translated XLF for import

Note: If English text changes during import, all translations for that segment will be cleared.
